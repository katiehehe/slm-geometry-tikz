# Golden Set — Spec-First Geometry → TikZ

150 curated, verified examples. Each one is a coordinate-free geometry scene
(plain text, no coordinates given) paired with a single TikZ/PGF figure that
draws it. This is the **v2 construction target**: the label emits a
coordinate-free construction and lets PGF compute the coordinates at compile
time (the representation that took the hardest construction from 0.02 → 0.98).

## Why it's "golden"

Every row passes the project's own falsifiable gate:

1. **figure-only** — starts with `\begin{tikzpicture}`, ends with `\end{tikzpicture}`, no prose.
2. **compiles** — builds to a PDF under `tectonic`.
3. **coordinates correct** — every named point lands within `0.05` of the ground-truth construction.

The rows are generated by the same seeded engine as the full dataset
(`scripts/build_golden_set.py`), so labels are correct by construction, then
each candidate is independently compiled and coordinate-checked before selection.
Candidate verified rate this build: **450/450 = 1.000**. The written file was
re-checked: **150/150** figure-only + coordinates-correct. The set is disjoint
from the held-out eval sets (`data/eval.jsonl`, `data/eval_pgf.jsonl`).

## Composition (150 rows)

- **Operations** (final + setup steps): foot-of-altitude 55, line-intersection 34, plus reflection, midpoint, and point-on-circle. Weighted toward the hard ops the frontier sweep found unreliable, with an easy/short/round tail so the set shows the whole behavior.
- **Chain length** (derivation steps): c2 = 20, c3 = 35, c4 = 70, c5 = 25.
- **Numbers**: 120 irregular, 30 round. (Round numbers are fakeable; irregular numbers force real computation.)

## Schema (one JSON object per line)

| field | type | meaning |
| :-- | :-- | :-- |
| `constraints` | list[str] | the scene as individual relationship statements |
| `description` | str | the constraints joined into one paragraph (the model input) |
| `tikz` | str | ground-truth figure (the label) |
| `points` | {name: [x, y]} | ground-truth coordinates used for grading |
| `chain` | int | number of derivation steps that compose |
| `irregular` | bool | irregular vs round numbers |
| `tags` | list[str] | operations used in the scene |
| `id` | int | row index |

## Files

- `golden_set.jsonl` — raw records (for grading).
- `golden_set_chat.jsonl` — system/user/assistant chat format (for SFT).
